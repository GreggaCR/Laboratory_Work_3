# Лабораторная работа №3: Дроби и матричные операции

## Описание задачи
Проект посвящен разработке библиотеки для работы с матрицами, элементы которых представлены рациональными числами (дробями). Для обеспечения абсолютной точности вычислений при решении СЛАУ и обращении матриц использован тип данных `mpq_t` библиотеки **GMP**.

## Реализованный функционал
Библиотека включает в себя широкий спектр операций линейной алгебры:

### Матричное умножение:
* Умножение матрицы на вектор.
* Классический алгоритм умножения матриц ($O(n^3)$).
* Блочный алгоритм умножения (оптимизация работы с кэшем).
* **Алгоритм Штрассена**: Рекурсивный метод быстрого умножения матриц.

### Линейные системы и матричный анализ:
* Решение систем линейных алгебраических уравнений (СЛАУ) методами:
    * **Метод Гаусса** (прямой ход).
    * **Метод Жордана-Гаусса** (полное исключение).
* Нахождение обратной матрицы методами Гаусса и Жордана-Гаусса.

## Зависимости и сборка
Проект требует установленной библиотеки **GMP**.
